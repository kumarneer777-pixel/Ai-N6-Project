# Gym Trainer API

A simple backend API built using Node.js and Express.

## 🚀 Features
- Basic API setup
- JSON response system
- Ready for expansion (users, workouts, schedules)

## 📦 Tech Stack
- Node.js
- Express.js

## ▶️ Run Locally

1. Install dependencies:
npm install

2. Start server:
node server.js

3. Open in browser:
http://localhost:3000

## 📌 Purpose
This project is created as a base backend for a gym trainer system where trainers can manage clients, workouts, and schedules.
